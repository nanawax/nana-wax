Gestion des prix

Modifiez ci-dessous. L'aperçu de la boutique se met à jour en direct. Copiez ensuite le code pour publier.

Pour publier vos changements :
1. Touchez « Copier le code ».
2. Sur GitHub, ouvrez le fichier index.html et touchez le crayon ✏️.
3. Sélectionnez tout le bloc entre les lignes DÉBUT DES DONNÉES et FIN DES DONNÉES (ces deux lignes comprises) et remplacez-le par ce que vous avez copié.
4. Touchez Commit changes. En 1–2 minutes, les nouveaux prix sont en ligne pour vos clientes.
